# Driver for my Drive

A landing page for an automated Google Drive organization service that files documents into the right folders by subject — every hour, hands-free.

## Overview

This repository contains the landing page for Driver for my Drive, an automation service that organizes Google Drive files automatically. The service reads files in your Drive, categorizes them by subject matter, and moves them into appropriately named folders without any manual effort.

## Features

The landing page showcases:

- **Automated organization** - Files are sorted every hour based on their content
- **Zero-effort filing** - Users continue saving files normally; organization happens in the background
- **Visual before/after comparison** - Shows the transformation from messy to organized Drive
- **Social proof** - Real testimonials and live metrics
- **Clear call-to-action** - Email-based contact form for interested users

## Technical Details

- **Single-page HTML application** with embedded CSS
- **No dependencies** - Pure HTML/CSS implementation
- **Responsive design** - Mobile-friendly layout using CSS Grid and Flexbox
- **Custom styling** - Professional dark teal color scheme with accent highlights
- **Google Fonts integration** - Uses Playfair Display for headings
